Sistema para controle de medicamentos, criado como projeto de BootCamp. A aplicacao tem versao CLI em Python, versao web, integracao com a API publica ViaCEP, testes automatizados, CI com GitHub Actions e deploy no GitHub Pages.

- Repositorio: [github.com/HeberMacedo/medsync](https://github.com/HeberMacedo/medsync)
- Deploy: [hebermacedo.github.io/medsync](https://hebermacedo.github.io/medsync/)
- Tecnologias: Python, Pytest, Ruff, GitHub Actions, HTML, CSS, JavaScript e ViaCEP
